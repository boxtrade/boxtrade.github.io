## [basePath](http://localhost:51340/#/zh-cn/configuration?id=basepath)

- 类型：`String`

文档加载的根路径，可以是二级路径或者是其他域名的路径。

```
window.$docsify = {
  basePath: '/path/',

  // 直接渲染其他域名的文档
  basePath: 'https://docsify.js.org/',

  // 甚至直接渲染其他仓库
  basePath:
    'https://raw.githubusercontent.com/ryanmcdermott/clean-code-javascript/master/',
};
```
